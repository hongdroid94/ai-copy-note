### 주요 UI 요소 설명

#### 상단 네비게이션
- **로고 + 검색바**: 전역 검색
- **사용자 프로필**: 설정, 로그아웃

#### 왼쪽 사이드바 (250px)
- **카테고리 목록**: AI 자동 분류된 카테고리별 필터
- **태그 클라우드**: 자주 사용되는 태그
- **통계**: 각 카테고리별 메모 개수

#### 메인 영역
- **메모 입력 박스** (상단 고정)
  - 텍스트 입력 영역 (자동 확장)
  - 저장 버튼
  - AI 자동 분류 토글
- **메모 리스트** (카드 형식)
  - 제목 (AI 자동 생성 또는 사용자 입력)
  - 태그 뱃지
  - 내용 미리보기 (2줄)
  - 작성 시간
  - 액션 버튼 (즐겨찾기, 편집, 삭제)

## 8. 인터랙션 및 애니메이션

### 8-1. 메모 카드 호버 효과
- 마우스 오버 시 약간 확대 (scale: 1.02)
- 그림자 강화
- 액션 버튼 표시

3. AI 분석 로딩 스피너

- 부드러운 페이드 아웃/인 전환
